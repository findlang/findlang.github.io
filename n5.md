| 句型              | 时态 | 肯定（简体）    | 否定（简体）       | 肯定（敬体）     | 否定（敬体）           |
| --------------- | -- | --------- | ------------ | ---------- | ---------------- |
| **名词句**         | 现在 | 私は学生だ     | 私は学生ではない     | 私は学生です     | 私は学生ではありません      |
|                 | 过去 | 私は学生だった   | 私は学生ではなかった   | 私は学生でした    | 私は学生ではありませんでした   |
| **动词句**         | 现在 | 本を読む      | 本を読まない       | 本を読みます     | 本を読みません          |
|                 | 过去 | 本を読んだ     | 本を読まなかった     | 本を読みました    | 本を読みませんでした       |
| **形容词句（い形容词）**  | 现在 | りんごは赤い    | りんごは赤くない     | りんごは赤いです   | りんごは赤くないです       |
|                 | 过去 | りんごは赤かった  | りんごは赤くなかった   | りんごは赤かったです | りんごは赤くなかったです     |

## 日语动词「原形」与「ます形」系统对比表

| 分类        | 功能 / 含义                   | 构成规则                                                         | 常见相关句型 / 例子                                                                                                                           | 备注               |
| --------- | ------------------------- | ------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------- | ---------------- |
| **动词原形**  | 最基础形态，辞书形；用于非敬语场合、辞典、语法接续 | 一类、二类、三类动词原型（例：書く・食べる・する）                                    | - まえに（例：寝るまえに歯を磨く）<br>- ことです（例：毎日運動することです）<br>- ことがある（例：日本へ行ったことがある）<br>- ことができる（例：日本語を話すことができる）                                      | 表达一般事实、习惯、能力、经验等 |
|           | 表示尊敬（原形→ます形变化）            | 同下                                                           | —                                                                                                                                     | 进入敬体体系           |
| **动词ます形** | 敬体形式；用于礼貌表达               | 一类动词：う段→い段＋ます<br>二类动词：去る＋ます<br>三类动词：する→します；来る→きます            | - たい／ほしい（例：日本へ行きたい）<br>- ながら（例：音楽を聞きながら勉強します）<br>- ませんか（例：一緒に行きませんか）<br>- ましょう（例：始めましょう）<br>- ましょうか（例：手伝いましょうか）<br>- に行きます（例：買いに行きます） | 表示礼貌、愿望、提议、共同行动等 |
|           | 重要作用                      | 「ます形」＋こと → 名词化（例：勉強する→勉強します→勉強すること）<br>与其他词构成新词（例：食べます→食べもの） | 名词化、词汇扩展                                                                                                                              |                  |

## 日语动词三类分类与判断总表

| 类别              | 判断方法（口诀）                         | 基本形结尾             | ます形变化规则           | 例子                                       | 中文释义    | 记忆小技巧                                          |
| --------------- | -------------------------------- | ----------------- | ----------------- | ---------------------------------------- | ------- | ---------------------------------------------- |
| **一类动词（五段动词）**  | **结尾是「う段假名」**（除了「いる」「える」结尾的特殊动词） | う、く、ぐ、す、つ、ぬ、ぶ、む、る | 把最后的「う段」改成「い段」＋ます | 書く→書きます<br>飲む→飲みます<br>話す→話します<br>待つ→待ちます | 写、喝、说、等 | 🎯口诀：<br>结尾假名在う段 → 一类<br>（例：く→きます）             |
| **二类动词（一段动词）**  | **结尾是「いる」「える」且前面是「い段/え段」假名**     | いる／える             | 去掉「る」＋ます          | 食べる→食べます<br>見る→見ます<br>起きる→起きます           | 吃、看、起床  | 🍎口诀：<br>看见“い/え + る”，多数是二类<br>如：たべ**る**、み**る** |
| **三类动词（不规则动词）** | **只有两个！**                        | する・くる             | 特殊变化              | する→します<br>くる→きます                         | 做、来     | 🌟口诀：<br>“做事”和“来去”最特别，只记这俩就行                   |

## 初学者判断速记法

| 步骤                     | 判断要点                          | 举例        | 结论   |
| ---------------------- | ----------------------------- | --------- | ---- |
| **① 看结尾是不是「る」**        | 如果不是「る」，100%是一类动词             | 書く、飲む、話す  | ✅ 一类 |
| **② 是「る」结尾？看前一个假名**    | 是「い段」或「え段」 → 二类动词             | 食べる、見る    | ✅ 二类 |
| **③ 特殊情况：い/え段＋る 但属一类** | 帰る（かえる）、走る（はしる）、要る（いる）、知る（しる） | ❗️记成例外：一类 |      |
| **④ 特殊动词**             | する、くる 及其派生（勉強する、電話する）         | ✅ 三类      |      |

## 🟠 一、「て形」——表示中顿、并列、动作状态、轻微因果关系
